LIPLEY CUSTOM FONTS
====================

Place your custom "Hitalica" font files in this folder.
For maximum browser compatibility, please upload the following formats:
- Hitalica.woff2
- Hitalica.woff
- Hitalica.ttf

By default, the website's CSS is configured to check for:
- "assets/fonts/Hitalica.woff2"
- "assets/fonts/Hitalica.woff"

If the font is not found, the website will fall back to a premium serif typeface ('Playfair Display' or 'Didot') for the LIPLEY branding elements, while all other text uses 'Plus Jakarta Sans'.
